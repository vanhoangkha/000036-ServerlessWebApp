# Mô-đun 2: Xác thực và Đăng kí User với Amazon Cognito User Pools

Ở mô-đun này, bạn sẽ tạo một [Amazon Cognito][cognito] user pool để quản lý các tài khoản user. Bạn sẽ deploy các trang liên quan để cho phép khác hàng của bạn có thể đăng ký user mới, xác nhận email và đăng nhập vào trang web.
