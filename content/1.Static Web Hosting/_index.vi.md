# Mô-đun 1: Host Web tĩnh với AWS Amplify Console

Ở mô-đun này bạn sẽ cấu hình AWS Amplify Console để host những tài nguyên tĩnh cho ứng dụng web của bạn.Ở các mô-đun tiếp theo bạn sẽ thêm các chức năng động cho các trang của bạn sử dụng JavaScript để gọi các RESTful APIs xây dựng với AWS Lambda và Amazon API Gateway.
